語言特性

為台語設計的程式語言

完整的現代程式語言功能，以台羅拼音為語料庫，讓台語在數位時代活下去。

🔤
台羅拼音關鍵字
所有保留字均使用台灣閩南語羅馬字，包含聲調符號，忠實呈現台語發音與書寫。
🔒
強型態系統
整數、浮點數、字串、布林、陣列五種型態，執行期型態安全，不做隱性數值轉換。
⚡
雙執行模式
直譯器直接走訪 AST；編譯器將程式碼轉為 TaiVM 位元組碼，再由堆疊虛擬機執行。
🗑️
引用計數 GC
以 Python 物件引用計數為基礎，自動記憶體管理，無需手動釋放，循環引用由 Python 處理。
🏗️
TaiVM 堆疊虛擬機
自製堆疊機，支援函式呼叫框架鏈、靜態作用域環境查找，附反組譯工具查看位元組碼。
🌐
一等公民函式
函式可存為變數、傳遞引數。靜態作用域（Lexical Scope）確保閉包語意正確。

程式範例

看台文程式怎麼寫

從 Hello World 到遞迴函式，完整示範台文語言的特性。

hello.taibu
// 台文的第一個程式
kóng "lí hó, Sè-kài!" ;
kóng "哈囉，世界！" ;

tòng-tsò miâ = "台灣" ;
kóng "lí hó, " + miâ + "!" ;
fibonacci.taibu
// 遞迴函式：費波那契數列
hàm-sò fibonacci(n) {
    nā-sī n bô-khah-tuā 1 {
        tò-tńg n ;
    }
    tò-tńg fibonacci(n - 1) + fibonacci(n - 2) ;
}

se̍h-lin-long i = 0 kàu 9 {
    kóng "F(" + i + ") = " + fibonacci(i) ;
}
factorial.taibu
// tng（while）迴圈計算階乘
hàm-sò kai-sǹg(n) {
    tòng-tsò kiat-kó = 1 ;
    tòng-tsò i = 1 ;
    tng i bô-khah-tuā n {
        kiat-kó = kiat-kó * i ;
        i = i + 1 ;
    }
    tò-tńg kiat-kó ;
}

se̍h-lin-long n = 1 kàu 10 {
    kóng n + "! = " + kai-sǹg(n) ;
}
control_flow.taibu
// kè-sio̍k (continue) 與 tn̄g-khì (break)
se̍h-lin-long i = 1 kàu 10 {
    nā-sī i % 2 pênn-pênn 0 {
        kè-sio̍k ;   // 偶數：繼續下一圈
    }
    kóng i ;  // 印出奇數
}

// nā-sī / nā-bô 多條件
tòng-tsò x = 42 ;
nā-sī x khah-tuā 100 {
    kóng "大數" ;
} nā-bô nā-sī x khah-tuā 50 {
    kóng "中數" ;
} nā-bô {
    kóng "小數（" + x + "）" ;
}
正式語法

EBNF 語法定義

台文程式語言的形式文法，以 Extended Backus–Naur Form 表示，運算子優先序由低到高。

grammar.ebnf（節錄）
program     = { statement } EOF ;

statement   = var_decl | if_stmt | while_stmt | for_stmt
            | func_decl | return_stmt | break_stmt
            | continue_stmt | print_stmt | block | ... ;

var_decl    = "tòng-tsò" IDENT "=" expression ";" ;
if_stmt     = "nā-sī" expression block [ "nā-bô" ( if_stmt | block ) ] ;
while_stmt  = "tng" expression block ;
for_stmt    = "se̍h-lin-long" IDENT "=" expression "kàu" expression block ;
func_decl   = "hàm-sò" IDENT "(" [ param_list ] ")" block ;

comp_op     = "pênn-pênn" | "bô-pênn" | "khah-sè"
            | "khah-tuā" | "bô-khah-tuā" | "bô-khah-sè" ;

(* 優先序：ia̍h-sī < kah < m̄-sī < 比較 < +-  < */% < 單元 < 後綴 *)
開始使用

安裝與執行

只需 Python 3.9+，無需安裝任何第三方套件。四種執行模式任君選擇。

01
下載專案
從 GitHub 取得完整原始碼。
git clone https://github.com/taibu-lang/taibu
cd taibu
02
直譯模式
直接執行 .taibu 原始碼，即時輸出結果。
python taibu.py run examples/hello.taibu
03
編譯 + 執行
編譯為 TaiVM 位元組碼（.tbc），再執行。
python taibu.py compile hello.taibu
python taibu.py exec hello.tbc
04
互動模式 REPL
啟動互動式命令列，即時測試程式碼片段。
python taibu.py repl
