# 7.1 管道（Pipe）

## 匿名管道

`pipe()` 建立一個單向資料通道，回傳兩個檔案描述符：

- `fd[0]`：讀取端（read end）
- `fd[1]`：寫入端（write end）

```
pipe(fd) 後：
  fd[1] (寫) ──────────→ fd[0] (讀)
```

### 範例：父子程序透過 Pipe 通訊

```c
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>
#include <string.h>
#include <sys/wait.h>

int main() {
    int fd[2];
    if (pipe(fd) == -1) { perror("pipe"); return 1; }

    pid_t pid = fork();
    if (pid < 0) { perror("fork"); return 1; }

    if (pid == 0) {
        // ── 子程序：讀取 ──
        close(fd[1]);  // 關閉寫入端
        char buf[256];
        ssize_t n = read(fd[0], buf, sizeof(buf) - 1);
        if (n > 0) {
            buf[n] = '\0';
            printf("[子程序] 收到：%s\n", buf);
        }
        close(fd[0]);
    } else {
        // ── 父程序：寫入 ──
        close(fd[0]);  // 關閉讀取端
        const char *msg = "Hello from parent!";
        write(fd[1], msg, strlen(msg));
        printf("[父程序] 傳送：%s\n", msg);
        close(fd[1]);
        wait(NULL);
    }
    return 0;
}
```

**編譯執行：**

```bash
$ gcc -o pipe_demo pipe_demo.c && ./pipe_demo
[父程序] 傳送：Hello from parent!
[子程序] 收到：Hello from parent!
```

### Pipe 注意事項

1. **單向**：一個 pipe 只能單向傳遞資料。如需雙向，需要兩個 pipe
2. **關閉未使用的端**：父程序關閉讀取端，子程序關閉寫入端
3. **阻塞行為**：讀取空 pipe 會阻塞；寫入滿 pipe 也會阻塞
4. **EOF 條件**：當所有寫入端被關閉，讀取端 read() 回傳 0

## 命名管道（FIFO）

FIFO 透過檔案系統建立，讓**無關的程序**也能通訊：

```bash
# 終端機 A
$ mkfifo mypipe
$ cat > mypipe
Hello! (輸入後按下 Enter)

# 終端機 B
$ cat mypipe
Hello!
```

### C 程式使用 FIFO

```c
// 寫入端
#include <stdio.h>
#include <fcntl.h>
#include <unistd.h>

int main() {
    mkfifo("/tmp/myfifo", 0644);
    int fd = open("/tmp/myfifo", O_WRONLY);
    write(fd, "Hello via FIFO!\n", 16);
    close(fd);
    unlink("/tmp/myfifo");  // 刪除 FIFO
    return 0;
}

// 讀取端：類似，使用 O_RDONLY 開啟
```

## Shell 中的 Pipe

Shell 中 `|` 運算子就是 pipe 的經典應用：

```bash
ls -la | grep ".c" | wc -l
```

每個 `|` 創建一個 pipe，左邊命令的 stdout 連接到右邊命令的 stdin——這是由 Shell 透過 `dup2()` 重定向檔案描述符完成的。

## 思考題

> **Q**：為什麼 pipe 被稱為「單向」通訊？如何實作雙向通訊？  
> **A**：pipe 的資料只能從寫入端流向讀取端。要實作雙向通訊，需要兩個 pipe——一個用於 A→B，另一個用於 B→A。或者使用 Unix Domain Socket（`socketpair()`），它提供雙向的串流。
