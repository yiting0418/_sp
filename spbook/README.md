# 系統程式設計：從原理到實作

> Systems Programming: From Principles to Practice

一本以 **C/C++** 為核心，結合理論與實戰的系統程式設計教材。

---

## 本書簡介

系統程式（Systems Programming）是計算機科學中最核心、也最貼近硬體的領域。與應用程式開發不同，系統程式著重於如何讓電腦本身運作——管理處理器、記憶體、程序、以及如何把人類寫的程式碼轉換成機器可執行的指令。

本書涵蓋 **五大核心主題**：

| 主題 | 章節 | 說明 |
|------|------|------|
| 作業系統原理 | 第 1 章 | OS 角色、使用者/核心模式、系統呼叫、中斷處理 |
| 程序與執行緒 | 第 2 章 | fork、執行緒、Mutex 同步、競態條件 |
| 記憶體管理 | 第 3 章 | 記憶體佈局、Stack/Heap、malloc/free、常見錯誤 |
| 編譯器與直譯器 | 第 4 章 | 編譯流程、詞法分析、語法分析、遞迴下降 |
| 綜合實作 | 第 5 章 | 迷你 Shell、互動終端機 |

以及 **三個進階主題**：

| 主題 | 章節 | 說明 |
|------|------|------|
| 檔案系統與 I/O | 第 6 章 | 檔案描述符、read/write、目錄操作、inode |
| 行程間通訊 (IPC) | 第 7 章 | Pipe、FIFO、共享記憶體、信號量 |
| 網路程式設計 | 第 8 章 | Socket、TCP Server/Client、select 模型 |

## 開發環境

- **作業系統**：Linux（建議 Ubuntu 22.04+）或 WSL2
- **編譯器**：GCC 11+ 或 Clang 14+
- **工具建議**：Valgrind（記憶體檢測）、GDB（除錯）、strace（系統呼叫追蹤）
- **選擇性**：Docker（隔離環境）、VSCode + Remote WSL

## 原始碼

本書所有 C 語言範例程式碼皆可在各章節資料夾中找到 `.c` 原始檔。每個範例都經過測試，可直接編譯執行。
